## Teslo API

1. Colnar proyecto
2. ```npm install```
3. Clonar el archivo ```.env.teamplate``` y renombrar a ```.env```
4. Cambiar las variables de entorno
5. Levantar la base de datos
```bash
$ docker-compose up
```
6. Levantar ```npm run start:dev```
